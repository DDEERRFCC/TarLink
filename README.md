# TarLink

TarLink is a student internship management system designed to connect students, companies, and administrators on a single platform.  
It helps manage internship applications, job postings, student profiles, and progress tracking efficiently.

---

## 📌 Features

### 👨‍🎓 Students
- Create and manage student profiles
- Browse available internship opportunities
- Apply for internships
- Track application status
- Submit reports and feedback

### 🏢 Companies
- Create and manage company profiles
- Post internship opportunities
- Review student applications
- Accept or reject applicants

### 🛠️ Administrators
- Manage users (students & companies)
- Monitor internship progress
- Generate reports
- Maintain system data integrity
